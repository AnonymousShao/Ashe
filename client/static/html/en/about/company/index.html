Why did you choose the cloud

UnitedStack cloud was established in February 2013, is committed to global unified open cloud platform data center.

UnitedStack have cloud is the earliest it open source cloud computing company in China, is now complete C round of financing.

UnitedStack have Cloud is the first in the Chinese market to provide hosting a Private Cloud (Managed Private Cloud) service Cloud company, is also the first to provide public and China Cloud computing nodes hosting a Private Cloud area completely consistent high reliable Cloud Cloud services company, in May 2015, became the first interoperability through it foundation certification Cloud computing service providers in China. In the same year, selected Forrester China enterprise public Cloud Wave, will be the only open source Cloud platform; won 2015 of the American journal of CIOReview the world's 20 most development potential solution provider recommended, is the only Asian companies.

UnitedStack have cloud core products are highly reliable platform UOS, the platform focused on the elastic computing, distributed storage and IaaS core technology such as software defined network (SDN) and ability, in seconds deployment, unique features, such as online migration to cloud computing market in China to bring a new generation of safe, reliable, high-performance infrastructure cloud environment. In March 2016, the official launch of UOS version 3.0, it puts forward the model of‘ open source projects + professional service’, the standard of it services and UnitedStack have cloud new development platform, a new version of the unified deployment suite and files related to it system to provide, for the standardization of cloud computing, lay solid foundation for large-scale application.

UnitedStack cloud is now the Chinese market important hybrid cloud providers, its cloud services on the Internet, energy, finance, manufacturing and other industries received praised by users.
