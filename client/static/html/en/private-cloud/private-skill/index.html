UOS hosted cloud summary function

Elastic cloud host

UStack cloud provides rich cloud components, and provides the corresponding API interface, you can like building blocks to build its own cloud data center, and through the API elastic cloud computing resource scheduling.

Software defined network

Based on Neutron, OVS, VXLAN, NameSpace, such as technology, UStack hosted cloud will all physical world network device is designed to become operational network object.Only need the UStack hosting cloud management panel, can according to the business need to be flexible to quickly build complex network environment, more intelligent network management.

Unified storage

Unified the CEPH distributed storage, reduce the cost of cloud storage subversion native Open - stack storage architecture, reduce the data transmission between components, let your cloud platform experience more smoothly.
