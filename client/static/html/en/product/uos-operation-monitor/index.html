UOS Operator

Monitor

All-round monitoring platform failure.

Alarm platform

The alarm classified summary, to receive alarm information.

Log platform

Centralized collection of log, archive, compression, backup.

Visualization

Provide multi-layer monitoring o&m view, layers of refinement.

All-round monitoring

Provides monitoring services from the system layer to the application layer, from hardware to software, covering core services such as computing, networking, and storage.

Data analysis

The platform generated log, alarm, monitoring data in-depth analysis and found potential problems.

Quick troubleshooting

Through the linkage of all-round monitoring can be layers of deep into the rapid positioning of the fault point.

Solve the following pain points

Comprehensive monitoring, effective monitoring of cluster status

UOS is a set of subsystems composed of a set of hardware and software as one of the distributed system, covering a wide range, you can effectively monitor the cluster status.

Timely and quickly locate the fault information

UOS system is complex, when the system fails, timely and quickly locate the fault information, can minimize the loss, UOS Operator provide cascading alarm, can quickly locate the fault information.

Reliability verification and fault drills

Cloud platform is the cornerstone of building IT, its stability and reliability is critical, UOS designed cloud platform reliability verification, from the cloud platform for multiple dimensions to be reliable.

Integration of multiple o&m tools, visual o&m, improve user experience

UOS Operator integrates multiple operation and maintenance tools to a management panel, you can visualize the o&m, improve o&m efficiency, and o&m experience.
