UOS Manager

Multi-platform

Supporting any open source platform.

Management

Through the Keystone to do a unified account management.

Approval

Custom multi-level approval to meet business process needs.

Ticket

Effectively manage, maintain and track questions.

Core OpenStack full coverage

Including compute, storage, network, authentication, scheduling, monitoring and other core functions.

Rich management functions

Including user management, resource management, service management, cluster management and other management functions.

Compatible

100% compatible with native OpenStack API. Supports fully autonomous definition of network architecture and topology, support for web panel / CLI / API operations.

Solve the following pain points

Extended

Modular architecture, support plug-ins, easy and fast expansion, easy to solve the problem of custom development.

Easy to use

In-depth understanding of OpenStack function operation, well designed, simplified operation, allowing users to quickly get started OpenStack.

Performance

SPA architecture, the ultimate performance optimization, smooth user experience.
