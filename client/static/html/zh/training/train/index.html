UnitedStack 有云培训特色课程
UnitedStack 有云培训基于能力角色，对技术开发或技术服务相关岗位进行角色定义，按能力进行分类，包括不限于：研发团队（包括DevOps团队）、技术服务团队、运维团队和实施与项目经理团队，并基于角色定义设定了包括 OpenStack Ops 认证课程、Solution Architect 认证课程、DevOps 认证课程和 Technical Architect 认证课程等四大类专业 OpenStack 技术认证体系，以及通识课程和专业课程等两大类计算机及云计算综合培训体系。方便学员自我定位，按照目标岗位的能力指定培训蓝图。

OpenStack Ops 认证课程
UnitedStack 有云是中国第一家推出规模性、商用化托管运维服务的 OpenStack 云服务商，基于3年多的专业 DevOps 经验推出的 DevOps 培训课程，帮助学员掌握较好的运维方法论，并熟悉 DevOps 语言及工具，熟悉项目的开发工程、工具链、了解架构及开发细节，并有能力将项目部署、运维等相关操作封装成安全、简单的工具或命令。

UnitedStack Ops 培训课程分为100系列、200系列、300系列、400系列四个级别，涵盖LinU 型基础、OpenStack 知识、报表与监控、自动化水平以及运维方法论。

DevOps 认证课程

UnitedStack 有云的 DevOps 培训课程分为两个大类:

持续集成系统 (Continuous Integration,CI)

持续交付 (Continuous Deliver,CD)

涉及故障追踪管理、版本控制、作业构建、作业守护、代码审查、服务器生命周期管理、任务编排、配置管理、软件打包和仓库管理等等。

UnitedStack DevOps 培训课程分为203、403、603三个级别。

UnitedStack 通识课程
开源云计算领域有一个重要的特点就是包罗万象，几乎将IT领域所有的技术都囊括其中，对人的知识深度和广度的要求极高，UnitedStack 有云通识课程就是帮助所有技术线的工程师为后续的高强度专业训练打牢基础。该课程适用所有技术线的工程师，包含研发、解决方案架构师、项目经理、运维、实施、技术服务等角色。

UnitedStack 通识培训课程分为100系列、110系列、120系列三个级别，涵盖计算机基础课程、Linux、C/C++、Python高级开发、软件测试与质量控制、企业运维与 DevOps 基础、开源文化等多个方面。
