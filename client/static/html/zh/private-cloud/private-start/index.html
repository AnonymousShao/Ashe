上线流程，定制服务

  • 试用 UOS 公有云

    在您决定使用 UOS 托管云服务之前，您可以先通过试用 UOS 公有云了解和评估 UOS 云服务平台 各项功能和性能指标，评估是否满足您对私有云的需求。

  • 容量规划

    根据您的业务需求，做合理的容量规则，以确保您的投入在适当的水平，并为日后业务发展提供足够的扩容能力。

  • 部署实施

    UnitedStack 技术团队帮助用户选择与 UOS托管云 最佳匹配的硬件和优质的机房，以保证 UOS 托管云 的 SLA 和高性能。

  • 硬件 机房准备

    UnitedStack 技术团队现场或者远程实现服务器上的 UOS 托管云安装部署。在硬件和机房条件准备就绪的情况下，30台机器的 UOS 托管云部署时间约为9个小时。

  • 验证交付

    部署完成之后，现场签收工单，同时 UnitedStack 技术团队向客户交：

    1、API、CLI 和 Web 控制台。

    2、帮助手册和文档，以及必要的培训。

  • 持续运维监控管理

    UnitedStack 技术团队将持续运维和管理，我们会做哪些事情：

    1、24×7不间断的自动化监控和检测，并且实时呼应任何异常情况。

我们不卖硬件，不卖软件，

而为您交付更稀缺、更有商业价值的云计算平台技术和运维服务。

常见问题

  • Q：与其他私有云方案相比，UOS 托管云有哪些差异化？

    A：a）UOS 托管云使用全分布式冗余架构，通过普通的x86硬件提供高可靠性的云平台。而传统的所谓和私有云方案仅仅是服务器虚拟化，基于单一存储，无法很好地保护及隔离资源。
    b）相比其它私有云实施方案仅侧重部署上线，而不注重生产维护，UOS 托管云 更关心平台的长期运维及持续的功能性能提升。

  • Q：为什么推 UOS 托管云 服务，你们的考虑是什么？

    A：云计算技术发展这么多年，虽然现在仍然是个高大上的服务和技术，但从我们对技术的观察来看，云计算的 IaaS 模式很快将变成一个成熟的、商品化的技术，公有云技术壁垒必将被打破。但是企业私有云运维服务能力仍然是个问题，因此我们向客户开放技术，通过提供专业的私有云托管服务来帮助用户真正进入云计算殿堂，享受云计算带来的便利。

  • Q：使用 UOS 托管云 企业还需要投入云平台的研发和运维人员吗？

    A：当然是不需要的，UOS 托管云 服务实现了对硬件服务器状态、云平台维护的智能化监控运维，对于云平台基础架构的我们通过严格的 SLA 进行保证。您的研发及运维人员不需再为基础架构而浪费时间，从而可以将有限的精力投放到真正产生效益的业务系统建设当中。

  • Q：UOS 托管云 适合什么类型的业务或用户？

    A：a）在现有的公有云平台上有较大投入的企业用户，如云主机使用量超过100台的用户。使用 UOS 托管云 成本更低，自主性更强。
    b）已经在基础设施建设，如：服务器、数据中心有较大投入的企业。使用 UOS 托管云 将帮助您充分整合现有的IT资源，优化投资成本。
