UOS 架构图
UnitedStack 通过领先的技术，打造基于 OpenStack 的强大云平台。
UOS 云平台采用超融合架构，在标准的x86服务器和通用网络设备的基础上，通过弹性计算、统一存储和 SDN 网络等技术，构建了涵盖云主机、云硬盘和 VPC 网络在内的完整云计算基础设施，为用户提供全功能的云计算平台。通过强大的运维监控系统和完善的持续集成体系，将为 UnitedStack 云平台不断增加新的功能，为用户持续交付新的价值。

我们不卖硬件，不卖软件，

而为您交付更稀缺、更有商业价值的云计算平台技术和运维服务。

UOS 托管云功能概要

弹性云主机

UStack 有云提供了丰富的云计算组件，并提供对应API接口，您可以像搭建积木一样构建自己云数据中心，并通过 API 弹性调度云计算资源。

软件定义网络

基于 Neutron、OVS、VXLAN、NameSpace 等技术,UStack 托管云将所有物理世界的网络设备设计成了可操作的网络对象。只需要 UStack 托管云管理面板,就可以根据业务需要灵活快速地构建复杂的网络环境,使网络管理更加智能。

统一存储

统一的 CEPH 分布式存储，降低云平台存储成本，颠覆原生 OpenStack 存储架构，减少组件间的数据传输，让您的云平台体验更加顺畅。

UOS托管云亮点

  • 高可用

    提供与 UStack 公有云相同的 SLA 服务保障，无论是底层架构，还是专业的运维团队，保证您的云时刻在线。

  • 安全

    UStack 有云提供了全方位的网络和系统安全保护,并通过稳固的底层构架保障用户数据的安全性和业务稳定性。

  • 监控

    UStack 有云对计算、存储、网络资源提供丰富的监控功能，还可以基于监控数据制定报警信息，提高业务系统运维的灵活性。

  • 管理

    有云的托管运维模式，由 UStack 专业的运维团队运维，减少维护的复杂度，降低了运维的人力成本和技术成本。

  • 升级

    UStack 有云为您提供持续不断的产品更新，优化云平台性能，上线新的产品功能，您的云平台将在技术和功能上保持领先。
